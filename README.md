# PROJECT 127 - LTO Management System

#### Application Description
This information system will be a "LTO Database Management System", designed to manage the driver's registration, violation, fees and such.

#### How to run the program (in Terminal)
1. Change line 9 in connect.py. Replace <yourpassword> with your actual MySQL root password.
2. Open a terminal in the root directory of your project and execute 'python main.py'
